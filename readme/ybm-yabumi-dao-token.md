## <mark style="color:blue;">veYBMとは？</mark> <a href="#veymwktoha" id="veymwktoha"></a>

veYBMとは、YBMをロックすることで得られる転送不可のトークンです。

最長4年のロックで 1 YBM ＝ 1 veYBMです。

1年ロックの場合は 1 YBM ＝ 0.25 veYBMです。

時間の経過で減衰していきます。

ロック期間はいつでも最長4年まで延長できますが、短縮や解除をすることはできません。
